Deze pagina is mijn moodboard pagina en dient als mijn Readme

Inspiratie

Mijn inspiratie kwam vooral van de basic sites die er vroeger waren waarop ik dan Flash games kon spelen.

Kleuren

Alle kleuren die ik heb gebruikt kunnen gevonden worden in het thema. Het blauw dient als referentie naar de zee, want het gaat over een piratenschip. Het bruin is dat van de planken die gebruikt worden voor een schip te maken. het groen/geel heb ik gebruikt omdat het de natuur en de gebouwen weergeeft. Wit en zwart zijn ook te vinden op de typische vlag van piraten. Het grijs/silver stelt de weg/de stad/silveren munten voor.

Font

Als font heb ik Bodoni Moda gekozen omdat het een font is dat mij wel een piratenvibe gaf, maar toch nog mooi past in het stadsgedeelte.

Javascript

William had mij een auto-type plugin laten zien en ik vond dat wel een leuke manier om bepaalde teksten te plaatsen. Ik het hiervoor (https://typeitjs.com/) gebruikt om bepaalde teksten automatisch uit te typen. Voor mijn Jquery heb ik iets heel subtiel, het maakt mijn checkboxen net iets mooier (http://icheck.fronteed.com/). Voor mijn formulier bij Buy heb ik me gebaseerd op de Discord login opdracht.
